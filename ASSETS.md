# Необходимые файлы для лендинга

Положите эти файлы в корень проекта (рядом с `index.html`), чтобы сайт отображался полностью.

## Обязательно

| Файл | Назначение |
|------|------------|
| `первый экран.jpg` | Фон первого экрана (desktop) |
| `первый экран для смартфонов.jpg` | Фон первого экрана (mobile) |
| `logo-dance-life.svg` | Логотип (можно заменить на оригинальный файл) |
| `Николас Кредедио.jpg` | Фото преподавателя |
| `Дмитрий Лиферов.jpg` | Фото преподавателя |
| `Григорий Рябцев и Милена Калинина.jpg` | Фото преподавателей |
| `school/01.jpg` … `school/09.jpg` | Галерея школы |
| `polozhenie-o-chempionate.pdf` | Положение о чемпионате |
